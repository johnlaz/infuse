The Tax You're Paying

shouldn't need a dev sprint.

Every copy tweak gets routed through code review, source control, and a deployment pipeline. For a comma.

// PROBLEM 01

The handoff trap

Designers ship a perfect file. Clients want "just one small change." Someone has to open a text editor, hunt for the string, push, deploy.

// PROBLEM 02

The image swap

Replacing a hero image means resizing, uploading to a CDN, rewriting paths, and praying nothing breaks. For a stock photo.

// PROBLEM 03

The rebrand

New brand colors mean grepping through CSS variables, hex codes, and forgotten inline styles. Eight hours for what should be a swatch.

Everything Inside

Built for rapid revisions.

Six core systems. One self-contained PWA. No accounts, no backend, no surveillance.

// 01Aa

Visual Content Editor

Every H1–H6, paragraph, link, button, list item, label, table cell, and blockquote becomes an editable card. Character counts, length warnings, tag badges, filter tabs.

// 02⊡

Live Preview

Full-width iframe with macOS-style chrome. 300ms debounce. Click any element in the preview to jump to its edit card.

// 03▣

Image Management

Upload from disk — no URLs. Embedded as base64 so the export is fully self-contained. Alt text editing for every image, every time.

// 04✦

Groq Deep Scan

One click finds images the DOM parser misses: CSS background-image values, inline styles, srcset, data-src. Newly found images appear with a ✦ badge.

// 05◐

Theme System

Three preset palettes that auto-map to your CSS variables. AI generates three custom brand-aware themes from your content. Manual color picker reads every variable from your style blocks.

// 06↗

PWA + Offline

Installs to your dock or home screen. Works offline after first load. GitHub Pages compatible out of the box with .nojekyll and 404 redirect baked in.
